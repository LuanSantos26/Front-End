Tabelas

As tabelas são usadas para exibir dados em formato tabular, com linhas e colunas. Elas são úteis para organizar informações de forma clara e estruturada.

As tabelas são compostas por linhas (tr), colunas (td ) e cabeçalhos (th). O elemento thead é usado para agrupar o cabeçalho da tabela, tbody para o corpo e tfoot para o rodapé.
